# Focus Timer

Focus Timer is a productivity-focused VS Code extension designed for developers who often lose track of time while coding.

This extension helps programmers maintain a healthy workflow by reminding them to take breaks, stay hydrated, relax, and avoid long continuous coding sessions.

# Usage
1. Install extension
2. Activate extension by clicking on icon from sidebar
3. Enter your timer (hours / minutes)
4. Start the timer


# Why This Extension?

Many developers unintentionally code continuously for several hours without:

- drinking water
- taking breaks
- stretching
- relaxing their eyes
- maintaining proper posture

Over long periods, this can negatively affect:

- physical health
- eye health
- posture
- productivity
- mental focus

Focus Timer helps solve this problem with simple visual reminders and time awareness.
